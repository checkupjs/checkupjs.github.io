Getting Started

Checkup was created to optimize writing tasks that enable you to gather the information that's useful to you, and use that information to make decisions. Checkup allows you to:

  • Create and manage Plugins
  • Write Tasks
  • Add Task Actions to provide calls to actions
  • Utilize the SARIF output in external tools

Checkup can be used through the command line, or programmatically through the Node API.

Installation#

Checkup has it's own CLI for running tasks, installable via npm.

  • npm
  • Yarn
npm i -D @checkup/cli
yarn add @checkup/cli --dev

The Node API, which provides a way to run tasks programmatically, can also be accessed through the @checkup/cli package.

Creating a configuration file#

Checkup uses a configuration file to define the plugins and tasks to run. You can create a configuration file by running the following command:

checkup generate config

This will generate a .checkuprc file in the current directory, with the following contents:

.checkuprc
{  "$schema": "https://raw.githubusercontent.com/checkupjs/checkup/master/packages/core/src/schemas/config-schema.json",  "excludePaths": [],  "plugins": [],  "tasks": {}}

Config file options#

  • excludePaths: A list of paths to exclude from analysis.
  • plugins: A list of plugins to load.
  • tasks: A list of tasks to run.

Command line#

Usage#

checkup --help

Run all configured Tasks in the current directory#

checkup run .

Run checkup using a custom configuration path or URL#

By path:

checkup run --config ./some/path/.checkuprc

By URL:

checkup run --config https://example.com/config.json

Run a single Task#

checkup run . --task my-task

Run Tasks by category#

checkup run . --category my-category

Run Tasks by group#

checkup run . --group my-group

Provide a custom format for output#

checkup run . --format checkup-formatter-pretty

Specify a different plugin directory to load plugins from#

checkup run . --plugin-base-dir ./my-plugin-dir

Generate a config file#

checkup generate config

Generate a plugin project#

checkup generate plugin my-plugin

Generate a task in a plugin directory#

checkup generate task my-task

Generate actions in a plugin directory#

checkup generate action my-action

  async run(): Promise<Result[]> {    // Gather and populate data into results
    return this.results;  }}
